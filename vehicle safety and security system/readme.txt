1. Vehicle safety and security system is based on PIC18F4520 microcontroller
2. Project contains GSM, GPS, Vibration sensor, Keypad and alarm
3. If any vibration detected while vehicle is in off, considering that was theft 
4. If any vibration detected while vehicle is in on, considering that was accident
5. when theft detcted message will be sended to the user.
6. if accident detected message will be send on relatives/doctors.
