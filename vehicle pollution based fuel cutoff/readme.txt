1. Vehicle pollution and insurance based fuel cutoff system is based on atmega328p (GCC)
2. This project contains smoke sensor, RTC, GSM, keypad and alarm
3. If the pollution is reached maximum user will be warned.
4. If incase the warning limit exceed vehicle fuel cutoff and registration number sended to RTO.
5. Same as insurance renewing, last data of insurance will be warned user through sms.
6. If he fail to renew insurance on time vehicle speed will be reduced to avoid accidents.
